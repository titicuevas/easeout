# EaseOut

**EaseOut** is a calming web journal where users can quickly ease out emotional tension by writing or recording voice notes. It's private, fast, and designed for those tough days when you just need to let it out.

## Features
- Voice and text journal entries
- Private and secure
- Visual, minimal UI
- Built with Laravel + React + Tailwind
